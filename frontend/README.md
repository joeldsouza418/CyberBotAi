# Frontend (React + Vite)

## Run
```bash
cd frontend
npm install
npm run dev
```

Open `http://localhost:5173`.

The UI sends requests to backend:
- `http://localhost:8000/api/v1/agent1/process`
- `http://localhost:8000/api/v1/agent2/domains`
- `http://localhost:8000/api/v1/agent2/chat`
- `http://localhost:8000/api/v1/agent3/score-domain`
- `http://localhost:8000/api/v1/agent3/report`
- `http://localhost:8000/api/v1/agent3/report/excel`

Use API key: `demo-key-123`.
